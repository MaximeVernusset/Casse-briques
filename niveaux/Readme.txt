1 brique par ligne :

- 1er chiffre  : type de brique (de 1 à 4 : faible, moyenne,
  forte, bonus) ;
- 2ème chiffre : position axe des x (de 0 à 19) ;
- 3ème chiffre : position axe des y (de 0 à 11).


Attention : ne pa laisser de sauts de ligne après l'édition du niveau.

Tant qu'il y a des niveaux à jouer, le programme peut les lire.
Il faut les nommer : "n°_du_niveau.lvl"
et les placer dans ce dossier ("niveaux").
